String handleIncomingMessage ( void  )

Messages from controller are handled by this function. Each message is a single instruction string. Instruction string gets broken into instruction code, id, and parameter. Passed in piecewise to <module>.set function. If a return message is generated from the <module>.set function, this function returns that message.
